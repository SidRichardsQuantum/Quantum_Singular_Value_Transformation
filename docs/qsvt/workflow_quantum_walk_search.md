# Quantum Walk Search Workflow

## Target

`quantum_walk_search_workflow(adjacency, marked_vertex, degree=...)`
demonstrates continuous-time quantum walk search on a finite graph.

## QSVT Idea

The search dynamics are generated by a marked-vertex Hamiltonian. Polynomial
approximations to the best-time propagator expose the same matrix-function
view used by QSVT-style signal transformations.

## Implementation

The workflow builds a dense search Hamiltonian from the graph adjacency matrix
and marked vertex, samples exact spectral evolution over a time grid, selects
the best marked-vertex probability, and fits polynomial cosine and sine
components for that time.

## Diagnostics

The result stores the graph, search Hamiltonian, marked vertex, hopping rate,
time grid, marked-vertex probabilities, best exact state, polynomial best-time
state, probability error, state error, operator error, and a resource proxy.

## Scope

This is a finite dense graph demonstration. It does not implement scalable
graph or marking oracles, amplitude estimation, optimized search circuits, or
hardware execution.

## API

```python
from qsvt.algorithms import quantum_walk_search_workflow

result = quantum_walk_search_workflow(adjacency, marked_vertex=0, degree=18)
report = result.as_report()
```
